# Medical Diagnosis using Bayesian Networks

## 📌 Project Description
This project implements a medical diagnosis system using Bayesian Networks. 
It predicts diseases such as Flu, Pneumonia, and COVID based on symptoms.

## 🛠 Technologies Used
- Python
- pgmpy
- Google Colab

## ⚙️ How to Run
1. Open the Colab notebook
2. Install pgmpy
3. Run all cells

## 📊 Sample Output
Flu: 0.7065  
Pneumonia: 0.1023  
COVID: 0.1892  
None: 0.002
